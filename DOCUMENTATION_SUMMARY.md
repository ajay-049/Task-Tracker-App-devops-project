## 🎯 Quick Start (Tl;dr)

### Absolute Fastest Way (5 min)

```bash
# 1. Setup database
mysql -u root -p < database/schema.sql
# Password: root

# 2. Terminal 1 - Backend
cd backend && npm install && npm start

# 3. Terminal 2 - Frontend
cd frontend && python -m http.server 8000

# 4. Browser
http://localhost:8000
```

**That's it!** ✅
